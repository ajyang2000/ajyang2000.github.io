TEAFENDERS (September 2020 to December 2020)

For the project of the class on Video Game Design, I worked along a team totaling of 12 students from the disciplines of Computer Science, Music, Art and Animation, to create a casual coop game, called Teafenders. Teafenders is a short defense game where 1-4 players protect teacups from living teabags for a given time. During the pre-alpha state of the project, I initially contributed to the mechanics (movement, jump and attacking) and implementing the animations (walk) of the players’ character. From the alpha version onwards, my sole responsibility was on the UI aspects of the game, both in the menus and in-game. These tasks include:
  • Completing the button functions for each menu
  • Loading the appropriate images when the character and level are selected. Also, I needed to ensure that the correct data is sent to the UI during gameplay.
  • Implementing the UI on each character’s models
  • Scaling the UI to adjust to screen size
  • Creating a save and load system for the scores
